# 🩺 Landing Page pour Médecin

Une landing page moderne et responsive développée avec **React.js**, **Tailwind CSS** et **Framer Motion**. Elle met en avant les services médicaux, le profil du médecin, les témoignages des patients ainsi qu'un appel à l'action pour la prise de rendez-vous.

## 🚀 Technologies utilisées

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🎬 Framer Motion
- ⚡ Vite

## ✨ Fonctionnalités

- Design responsive
- Animations fluides
- Interface moderne et intuitive
- Présentation du médecin
- Présentation des services médicaux
- Section À propos
- Témoignages des patients
- Section Contact et prise de rendez-vous

## 📦 Installation

```bash
git clone https://github.com/votre-utilisateur/doctor-landing-page.git

cd doctor-landing-page

npm install

npm run dev
```
